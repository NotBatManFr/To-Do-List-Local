## About The Project

<a href="https://to-do-list.page.gd/" target="_blank" rel="noopener noreferrer">To the App [To-Do]</a>

A local task management application designed to help you organize and track your to-do items with ease. Create tasks with detailed information, set priorities, assign due dates and times, and manage your productivity all from your browser using local storage.

The application stores all data locally on your device, ensuring your tasks remain private and accessible offline.

<p align="right">(<a href="#readme-top">back to top</a>)</p>

## Features

- **Create Tasks** ‚Äî Add new tasks with a simple form interface
- **Task Details** ‚Äî Include task title, description, due date, due time, and priority level (Low, Normal, High)
- **Edit Tasks** ‚Äî Modify existing tasks with an intuitive edit/save toggle
- **Delete Tasks** ‚Äî Remove tasks with confirmation to prevent accidental deletion
- **Local Storage** ‚Äî All tasks are saved to your browser's localStorage for persistence across sessions
- **Responsive UI** ‚Äî Clean, grid-based interface with visual feedback for user actions

## Getting Started

### Prerequisites

- Node.js and npm installed on your system
- A modern web browser with localStorage support

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/MriteshAdak/To-Do-List.git
   cd To-Do-List
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run run
   ```

4. Open your browser and navigate to the local server address (typically `http://localhost:8080`)

### Usage

1. Click **"+ Add List"** to create a new task form
2. Fill in the task details:
   - **Title** ‚Äî Task name
   - **Description** ‚Äî Additional details
   - **Due Date** ‚Äî When the task is due
   - **Due Time** ‚Äî Specific time for the task
   - **Priority** ‚Äî Select Low, Normal, or High
3. Click **Save** (üíæ) to store the task
4. Click **Edit** (‚úèÔ∏è) to modify an existing task
5. Click **Delete** (üóëÔ∏è) to remove a task (confirmation required)

All tasks are automatically saved to your browser's localStorage.
